# Backend Recruitment Test

## Installation

**Clone repository and go to repository directory** 
```bash
cp .env.example .env
```
```bash
composer i 
```
```bash
docker-compose build 
```
```bash
make up 
```

## Running Tests

To run tests, run the following command

```bash
make test 
```

Code coverage report (HTML)
```bash
make html 
```
